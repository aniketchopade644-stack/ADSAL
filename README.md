# 📘 Advanced DSL Practicals

This repository contains advanced data structure practical programs implemented as part of the **Advanced Data Structures Laboratory (DSL)**.

---

## 📂 List of Practicals

### 🔹 Practical 1: Expression Tree

* Construct expression tree from prefix expression
* Perform postorder traversal
* Delete entire tree

---

### 🔹 Practical 2: Dijkstra’s Algorithm

* Find shortest path between nodes
* Calculate minimum transmission time
* Display reachable stations within limit

---

### 🔹 Practical 3: Prim’s Algorithm

* Find Minimum Spanning Tree (MST)
* Connect all nodes with minimum cost
* Display total cost

---

## 🛠️ Technologies Used

* C++

---

## 🎯 Objectives

* Understand advanced data structures
* Learn tree and graph algorithms
* Implement shortest path and MST techniques
